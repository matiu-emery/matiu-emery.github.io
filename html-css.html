whats the difference between Margin, Border, Padding

Margin, Border, and Padding are all CSS properties used to create space around HTML elements Margin refers to the space outside of an element's border. It is used to create space between elements Border refers to the line or edge around an element. It is used to create a visible boundary around the element. Padding refers to the space between an element's content and its border. It is used to create space between an element's content and its border. In summary, Margin creates space outside of an element, Border creates a visible boundary around an element, and Padding creates space between an element's content and its border.
